# 📍 Local Finder Pro

A smart web app to find nearby hospitals, restaurants, and ATMs using real-time location and maps.

## 🚀 Features
- Live Map
- Nearby Search
- Emergency Mode
- Voice Search

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Leaflet.js
- Overpass API
